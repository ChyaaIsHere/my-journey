My Daily Day |

Keseharian saya sih hampir kebanyakan baca buku novel dan nonton YouTube sih.. Ya kalau udah bosen banget dirumah biasanya saya sempetin buat jalan jalan sebentar. Tapi biasanya kalau ada waktu senggang, Saya coba hal-hal baru seperti ngoding ini yang sebelumnya saya ngga pernah lakuin. Kadang kalau emang waktunya mepet dengan hari ujian praktik jurusan, biasanya belajar virtualisasi, praktik pakai alat jaringan langsung, atau ngerancang topologi di cisco sih.

| Pernah coding?

Saya pernah mencoba membuat sebuah coding berbentuk JSON, LUA, dan XML dari salah satu game irama atau yang biasa disebut game rhythm. Namanya Friday Night Funkin! Yah saya pernah membuat sebuah lagu cover di game tersebut. dan agar gameplay lebih seru, pastinya saya mencari ide agar mod yang aku buat ini terlihat asyik dimainkan oleh siapapun. Bahkan saya pernah menghabiskan waktu seharian untuk mencari tahu cara membuat sebuah mekanik atau script tambahan untuk UI atau Hud game agar menjadi lebih menarik. Walaupun menghabiskan banyak waktu, itu menjadi sebuah tantangan baru untuk-ku. "Apakah saya dapat menyelesaikan ini? atau akan berhenti sampai sini saja?" kata diri saya sendiri.
